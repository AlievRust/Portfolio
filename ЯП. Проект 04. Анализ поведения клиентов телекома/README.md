## Описание

Исследование предпочтений клиентов сотового оператора (выбор наиболее доходного тарифа) для корректировки рекламного бюджета. Проверка гипотез.


## Итог:

- проведён анализ поведения клиентов, использующих разные тарифы в разных городах (столица и остальные), в разрезе разных услуг и месяцев года
- подтвердили гипотезу о большей прибыльности одного из тарифов
- отвергли гипотезу о равенстве выручки клиентов из столицы и других регионов


## Навыки и инструменты:

Python, Pandas, Matplotlib, NumPy, SciPy, описательная статистика, проверка статистических гипотез, исследовательский анализ, визуализация данных, предобработка данных.
