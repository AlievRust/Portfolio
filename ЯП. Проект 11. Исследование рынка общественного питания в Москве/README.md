## Описание  

По заказу инвесторов, планирующих запуск кофейни, провести анализ рынка общепита Москвы. Подготовить визуализации и презентацию на основе отчета.


## Итог  

- осуществлена подготовка данных и предобработка
- проведён поиск и анализ закономерностей в данных с построением разнообразных визуализаций (структура заведений, анализ среднего чека, географическая концентрация с отоборажением на карте, режим работы, количество посадочных мест)
- сделана детализация результатов исследования применительно к открытию кофейни (географическое распределение кофеен, количество и востребованность круглосуточных кофеен, уровень цен по округам)
- оформлена [презентация](https://github.com/AlievRust/Portfolio/blob/main/%D0%AF%D0%9F.%20%D0%9F%D1%80%D0%BE%D0%B5%D0%BA%D1%82%2011.%20%D0%98%D1%81%D1%81%D0%BB%D0%B5%D0%B4%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5%20%D1%80%D1%8B%D0%BD%D0%BA%D0%B0%20%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B3%D0%BE%20%D0%BF%D0%B8%D1%82%D0%B0%D0%BD%D0%B8%D1%8F%20%D0%B2%20%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B5/project11_presentation.pdf)
